Module: gestorEstados

Modulo para crear un gestor de estado que permite almacenar y recuperar el estado en sessionStorage.
Source:
  • gestorEstados.js, line 1

Methods

(static) createState(initialState, storageKey, stateChangeCallback) → {object}

Parameters:
Name Type Description
initialState * El estado inicial para el gestor de estado.
storageKey string | null La clave para almacenar el estado en sessionStorage. Si es null, no se almacena en sessionStorage.
stateChangeCallback function | null Una función callback que se ejecuta cuando el estado cambia. Recibe el nuevo estado como argumento.
Source:
  • gestorEstados.js, line 23
Returns:
Un objeto que contiene métodos para obtener, establecer y registrar cambios en el estado.
Type
object
Example
//Ejemplo de creación:
const contadorElement = document.getElementById('contador');
const contadorState = createState(0, 'contadorState',(state) => {contadorElement.textContent = state;});

//Ejemplo de cambiar estado:
incrementarButton.addEventListener('click', () => {
    contadorState.setState(prevState => prevState + 1);
});
