Methods

(static) renderPage(route, containeropt, callbackopt)

Renderiza una página web en un contenedor especificado y ejecuta una función de devolución de llamada opcional.
Parameters:
Name Type Attributes Default Description
route string La ruta de la página a renderizar, sin incluir la extensión ".html".
container string <optional>
 "content" El ID del elemento contenedor donde se renderizará la página. El valor predeterminado es "content".
callback function | null <optional>
 null Una función de devolución de llamada opcional que se ejecutará después de renderizar la página.
Source:

Throws:
Si la página no se encuentra o si ocurre un error durante la carga.
Type
Error
Example
// Ejemplo de uso:
renderPage("page1", "content", (renderedHtmlElement) => {
    // Realiza acciones adicionales después de renderizar la página
    console.log("Página renderizada:", renderedHtmlElement);
});
